PROJECT 01
커스텀 ERP 도입 — 풀사이클 단독 수행
2023.07 → 2024.12
200여 개 예산 항목을 엑셀로 수기 관리하던 환경에서, Tibero DB/Nexacro 기반 커스텀 ERP를 요구사항 정의부터 최종 검수까지 단독으로 수행한 프로젝트입니다.
Phase 1 — 요구사항 정의
현업 부서 인터뷰 → 예산·회계 프로세스 흐름 파악. 기존 엑셀 관리의 문제점(중복 입력, 추적 불가, 결산 지연) 문서화.
Phase 2 — 계정과목 표준화
200개 예산 항목 재분류. 부서별·사업별 매핑 테이블 설계. 마이그레이션 계획 수립.
Phase 3 — 데이터 검수 (SQL)
SQL 쿼리로 마이그레이션 데이터 정합성 검증. 불일치 항목 식별 및 수정.
Phase 4 — 기능 검수 · 안정화
전사 기능 테스트 수행. 오류 리포트 관리. 관리자 권한으로 시스템 운영.
BEFORE
엑셀 수기 관리 → 분기 결산 5일 소요
부서별 집행 현황 파악 불가
오류 발생 시 추적 불가능
AFTER
ERP 실시간 조회 → 결산 2일 단축
부서별·사업별 집행률 즉시 확인
데이터 기반 의사결정 환경 구축
예산 집행률 88.5% 달성
PROJECT 02
IT 비용 최적화 — 외주 내재화 + 재계약 협상
2023.01 → 2026.02
네트워크·PC 장애 발생 시 매번 외주업체를 호출하는 구조에서, 반복 패턴을 분석하고 1차 대응을 내재화하여 비용을 구조적으로 절감한 사례입니다.
분석 — 장애 패턴 구조화
1년간 외주 호출 이력 분류. 전체의 60% 이상이 단순 반복 장애임을 확인.
실행 — 1차 대응 매뉴얼 + 직접 수행
유형별 대응 매뉴얼 정리. 직접 트러블슈팅 체계 구축. 외주 호출을 복잡 장애로 한정.
협상 — 재계약 조건 개선
복합기 렌탈, 전자계약, 본인확인 서비스 등 재계약 시 단가 재협상 진행.
BEFORE
단순 장애에도 매번 외주 출장비 발생
재계약 시 기존 단가 자동 갱신
AFTER
1차 대응 내재화 → 유지보수 비용 30% 절감
재계약 협상 → 연간 약 500만 원 추가 절감
연간 유지보수 비용 30% + 500만원 절감
