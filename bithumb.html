PROJECT 01
ERP Implementation — 관리회계 디지털 전환
2023.07 → 2024.12
엑셀 수기 관리 환경에서 커스텀 ERP(High-Q, Tibero/Nexacro 기반)를 요구사항 정의부터 검수까지 단독으로 도입한 프로젝트입니다. 여신·채무·이자 회계계정 체계를 직접 설계하고, 데이터 마이그레이션·사용자 교육까지 수행했습니다.
Phase 1 — Requirements & Analysis
현업 부서 인터뷰 → 기존 프로세스 분석 → 기능 요구사항 문서화. 여신·채무·이자 계정과목 체계 정의 및 표준화.
Phase 2 — Design & Data Migration
예산/회계 프로세스 재설계. 엑셀 → Tibero DB 마이그레이션. SQL 기반 데이터 정합성 검증.
Phase 3 — Testing & Stabilization
기능 테스트 → 수정 → 재검증 사이클 반복. 시스템 안정화 및 사용자 교육.
BEFORE
엑셀 수기 관리 → 데이터 정합성 미보장
계정과목 비표준 → 결산 지연
수동 데이터 추출 → 분석 병목
AFTER
DB 기반 관리회계 → 실시간 데이터 접근
회계계정 체계 구축 → 결산 2일 단축
SQL 직접 추출 → 분석 속도 향상
PM 단독 수행 · 회계계정 체계 구축 · 결산 리드타임 2일 단축
PROJECT 02
Budget Control & Process Optimization
2023.01 → 2026.02
연 3.6억 규모의 인프라 예산을 단독으로 편성·집행·분석하며, 8개 외주업체의 계약·정산·SLA를 총괄했습니다. 재계약 협상과 업무 내재화를 통해 비용 30%를 절감한 프로세스 개선 사례입니다.
Budget Planning & Execution
연간 3.6억 인프라 예산 편성. 월별 집행 현황 분석 및 보고. 집행률 88.5% 달성.
Vendor Management
소방·승강기·복합기·네트워크 등 8개사 계약·정산·SLA 총괄. 연간 프로세스 표준화.
Cost Optimization
재계약 협상(복합기·전자계약·본인확인) → 연 500만 원 절감. IT 트러블슈팅 내재화 → 외주 비용 30% 감소.
집행률 88.5% · 유지보수 비용 -30% · 연 500만 절감
