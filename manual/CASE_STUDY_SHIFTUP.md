# Case Study: Shift Up (시프트업) 지원

## 📂 Overview
* **일자**: 2026-01-20
* **포지션**: 총무 직군 (Asset Management / Contract)
* **결과**: 제출 완료 (V7 스타일 고수)

## 💡 Strategy: "Game Dev Literacy"
"게임 개발을 이해하는 총무"라는 컨셉으로 접근. 게임 회사라도 총무 직군은 "깔끔하고 정돈된(칼각)" 문서를 선호할 확률이 높다는 가설 하에 **V7 (A4 최적화, 공문서 스타일)**을 최종 선택함.

## 🚀 Experience Highlights
* V7 (공문서 스타일) + V3 (디자인) + Fusion (표지+이력서) 다양한 시도 수행.
* 최종적으로 가독성과 전문성을 강조한 V7 제출.

## 🔧 Key Lessons
* 직무의 특성(총무/회계)에 따라 디자인적 화려함보다는 문서의 신뢰도와 정돈됨이 더 중요할 수 있음.
* 사진은 `assets/` 폴더의 최신 파일을 자동 인식하도록 시스템화함.
