Project 01

CPC 핵심 역량의 원천

커스텀 ERP 도입 — 전사 데이터 취합·검증

2023.07 → 2024.12 · 18개월 · 단독 전담

엑셀 기반 수기 관리에서 커스텀 ERP(Tibero/Nexacro)로 전환. 사업팀·회계팀·개발사의 자료를 요청→취합→형식 확인→기한 내 전달하는 프로세스를 18개월간 반복 수행. 여신·채무·이자 회계계정 200개를 직접 설계하고, 데이터 마이그레이션 정합성을 검증했습니다.

자료 취합 18개월 계정 200개 설계 결산 -2일
PHASE 1 — 요구사항

부서별 인터뷰 → 프로세스 분석 → 기능 요구사항 문서화. 여신·채무 회계계정 체계 설계.

PHASE 2 — 마이그레이션

엑셀 → Tibero DB 이관. SQL 기반 데이터 정합성 검증. 부서별 자료 취합·검수 반복.

PHASE 3 — 안정화

기능 테스트 → 수정 → 재검증 사이클. 관리회계 전환 완료. 결산 리드타임 2일 단축.
