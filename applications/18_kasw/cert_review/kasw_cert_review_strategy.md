- 함께만드는세상에서 사업관리·정산·서류검증 경험 → 심사업무 전이 가능
- 꼼꼼한 행정처리 역량 (예산 3.6억 집행·외주 8개사 관리)

## 3. Go/No-Go

### ✅ Go — 전략적 최적 매칭

**Go 사유:**
- 사회복지사 1급 = 우대사항 정면 충족 (경쟁자 중 석사+1급 보유자 희소)
- 3개월 계약만료 = 비자발적 퇴사 → **실업급여 트리거** (이전 피보험기간 합산)
- 영등포구 = 광명에서 접근성 양호
- 모집 2명 = 경쟁률 완화

**리스크:**
- 퇴직금 없음 (3개월 < 1년)
- 급여 253만 (낮지만 실업급여 확보용으로 수용)

## 4. 포지셔닝

### 전략: "사회복지사 1급 + 행정·정산 실무 경험자"
- 사회복지 석사 + 1급 자격 → 자격관리 직무의 전문성 이해
- 함께만드는세상 3년: 예산집행·정산·서류검증 → 심사업무 즉시 투입 가능
