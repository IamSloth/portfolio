| JD 요건 | 프로필 매칭 | 등급 |

### 핵심 강점
- **사회적기업 전공 석사** = 사회적경제 직무 최적 학력 (경쟁자 중 희소)
- 비영리법인 3년 = "관련 분야 경력" + "사회적경제 기관 경력" 동시 충족
- 사업 행정(예산·회계·정산) + 기획(ERP 도입 PM) 모두 경험
- 사회복지사 1급 = 사회적경제 생태계 이해도 증명

## 3. Go/No-Go

### ✅ Go — 최상급 매칭

**Go 사유:**
- 석사 전공이 "사회적기업" = JD와 **직접 매칭** (이 수준의 매칭은 전체 지원 중 최고)
- 비영리법인 3년 행정·회계 = 담당업무 정확 일치
- 구로구 개봉동 = 광명에서 **가장 가까운** 근무지 (1호선 1정거장)
- 3개월 시용 → 연장 = 실업급여 전략 부합 (연장 시 퇴직금도 가능)

**리스크:**
- 급여 미공개 (센터 내규)
- "채용사이트 접수 원칙" — 어떤 사이트인지 확인 필요

## 4. 포지셔닝

### 전략: "사회적기업 석사 + 비영리 행정 3년, 이론과 현장을 겸비한 실무자"
