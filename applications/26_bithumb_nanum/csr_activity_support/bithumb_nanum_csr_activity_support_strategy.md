## #3 Go/No-Go

### ✅ **STRONG GO — 최상위 매칭**

**Go 근거:**
1. **사회복지사 1급 + 사회적기업 석사** → 우대 "사회복지/CSR/ESG 전공" 초과달성
2. **함께만드는세상 3년** → 사회공헌/공공기관 실무 경험 직결
3. **1종 대형 면허 + 스타렉스 운전 경험** → 필수 요건 "대형승합차 실제 운전" 완벽 충족
4. **총무 3년 = 행정/문서/커뮤니케이션 역량** → 담당업무 전부 커버
5. **전국 기동성** → 봉사활동 전국 출장 대응 가능
6. **경력 무관** → 경력 장벽 없음
7. **계약직 6개월** → 유저 전략 부합 (실업급여 수급 가능)

**약점:**
- 봉사활동 기획/운영 "직접" 경험은 약함 → "사회공헌 기관에서 행정 지원한 경험" 프레임
- 6개월이라 퇴직금 없음 (1년 미만) → 실업급여만 타겟
- 빗썸 소속 아닌 빗썸나눔 소속 → 기대 관리 필요
