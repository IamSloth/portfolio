## 4. 지원 컨셉 (Positioning)

### 메인 컨셉
> **"사회공헌의 기획부터 운영까지, 비영리 현장에서 다져진 실무형 인재"**

### 핵심 셀링 포인트 3가지

1. **전문 학위·자격**
   - 사회복지대학원 석사 (사회적기업 전공) → 사회적 가치 사업의 이론적 기반
   - 사회복지사 1급 → 시니어 금융취약계층 이해의 전문성

2. **비영리 법인 3년+ 운영 실무**
   - (사)함께만드는세상 경영지원부: 예산 3.6억 집행 관리, 외주업체 선정·관리, 안전점검 일정 조율
   - 사업 운영에 필요한 행정·예산·일정 관리의 풀사이클 경험

3. **IT 활용 + 데이터 역량**
   - ERP 도입 PM (요구사항 정의 → 검수까지 풀사이클)
   - SQL 쿼리로 DB 직접 조회, Excel 피벗/VLOOKUP
   - 사업 성과 데이터 정리·보고에 즉시 활용 가능

### 자기소개서 방향

**서사 흐름**: 사회복지 석사(사회적기업 전공) → 비영리 법인 실무 3년 → 시니어 금융취약계층을 위한 사회공헌 사업

- **지원 동기**: 사회적 가치를 추구해온 커리어의 연장선. 금융취약계층 지원이라는 구체적 미션에 공감
- **강점 어필**: 사업 운영 실무(예산·외주·일정), IT 활용(ERP·데이터), 사회복지사 자격
- **약점 방어**: 금융교육 직접 경험 없음 → "사회복지 전문성으로 대상자(시니어) 이해 + 운영 실무로 사업 관리" 구도
