## #3 Go/No-Go

### ✅ **STRONG GO**

**Go 근거:**
1. **Excel 고급** → JD "중급 이상" 대비 Pivot/VBA/Power Query 보유. 즉전력
2. **데이터 추출/가공 경험** → SQL + ERP 데이터 추출 일상 업무
3. **총무 = HR 인접 직무** → 근태관리·문서발급·직원 문의 대응 경험 직결
4. **전사 커뮤니케이션 허브** → 8개 외주 + 전부서 조율 3년
5. **6개월 이상 근무 가능** → 우대사항 충족
6. **기존 NC 지원 데이터 재활용** → 추가 비용 0

**약점:**
- 경력 0~1년 요건에 3년 경력 → 오버스펙 우려. "단기 계약직이라 즉전력 환영" 프레임
- HR 전담 경험 없음 → "총무로서 HR 인접 업무 수행" 방어
