## 지원동기 (100~3000자)

```
3년간 비영리법인 경영지원부에서 총무·IT지원 업무를 수행하며, HR과 가장 가까운 자리에서 일했습니다. 직원들의 근태 현황을 ERP에서 추출해 보고서로 정리하고, 휴가·휴직 신청을 접수·처리하며, 각종 제증명 발급 요청에 대응하는 일이 일상이었습니다.

그 과정에서 인사 데이터를 다루는 일에 흥미를 느꼈습니다. 단순히 숫자를 옮기는 것이 아니라, 근태 패턴을 분석하면 조직의 업무 부하가 보이고, 휴가 사용률을 추적하면 구성원의 컨디션이 읽힙니다. 이런 데이터가 HR 의사결정에 근거를 제공한다는 점에서, 인사 데이터 업무를 더 전문적으로 경험하고 싶다고 생각했습니다.

특히 엔씨소프트처럼 수천 명 규모의 조직에서 HR 데이터를 체계적으로 관리하는 경험은, 소규모 법인에서는 접하기 어려운 것입니다. 대규모 인원의 근태 모니터링, 인사 데이터 추출·가공, 구성원 문의 대응이라는 업무 범위가 제 기존 경험의 연장선에 있으면서도 한 단계 확장된 영역이라 판단하여 지원합니다.

Excel Pivot/VBA/Power Query와 SQL을 활용한 데이터 추출·가공 역량을 보유하고 있으며, 커스텀 ERP 도입 프로젝트를 단독 수행한 경험이 있어 새로운 HR 시스템에도 빠르게 적응할 수 있습니다.
```

## 추가질문: 차별화된 장점 (100~1000자)

```
Excel 활용 수준이 "중급 이상"을 넘어 고급 영역에 있습니다. Pivot Table로 대량 데이터를 즉시 요약하고, VBA 매크로로 반복 작업을 자동화하며, Power Query로 여러 소스의 데이터를 병합·정제하는 것이 일상적인 업무 방식입니다.

여기에 SQL을 결합할 수 있습니다. 전 직장에서 Tibero DB 기반 ERP를 도입하며 SQL로 데이터 정합성을 검증하고, 필요한 데이터를 직접 추출하는 업무를 수행했습니다. HR 시스템에서 인사 데이터를 추출·가공할 때, Excel만 쓰는 사람과 SQL까지 활용하는 사람의 업무 효율 차이는 상당합니다.

또한 총무로서 전부서 요청을 접수·조율하는 허브 역할을 3년간 수행했습니다. 하루에도 수십 건의 문의를 처리하며 체득한 커뮤니케이션 역량은, 사내 구성원 HR 문의 대응 업무에 바로 적용할 수 있습니다.
```
