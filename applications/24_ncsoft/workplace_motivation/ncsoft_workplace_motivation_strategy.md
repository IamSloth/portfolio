| 어린이집 운영 | ⭐ | 직접 경험 없음. **사회복지사 1급** + 복지 도메인 이해도로 방어 |
| MS Office | ⭐⭐⭐ | Excel/PPT 실무 활용 |

## 강점 요약

1. **총무 3년 경력** — 핵심 필수 요건 정확히 충족
2. **예산·정산·외주 관리** — JD 업무의 70%가 이 영역
3. **사회복지사 1급** — 어린이집·복지시설 도메인 이해도 차별화
4. **계약직** — 구직 전략과 완벽 부합

## 약점 (방어 필요)

1. **식당/어린이집 직접 운영 경험 없음** → "시설 운영 = 외주 관리 + 예산 + 자산 관리"로 프레임 전환
2. **이벤트 기획 경험 부족** → 간접 경험 발굴 필요
3. **게임 업계 경험 없음** → 총무직이라 도메인 특이성 낮음. 문제 없을 것

# 4. 지원 컨셉 (Positioning)

## 핵심 메시지

> **"총무 3년 + 사회복지사 1급 = 복지시설 운영의 실무+도메인을 동시에 이해하는 사람"**

## 포지셔닝 전략

| 축 | 전략 |
|----|------|
| **강점 극대화** | 외주업체 8개사 관리·예산 3.6억 집행 → "복지시설 운영의 핵심은 결국 외주 관리와 예산 통제" |
| **약점 전환** | 식당/어린이집 직접 경험 없음 → "시설 관리의 본질(외주·예산·자산)은 동일. 도메인만 다를 뿐" |
| **차별화** | 사회복지사 1급 → 어린이집 운영 관련 법규·제도 이해. 일반 총무와 다른 관점 |
| **동기 부여** | "복지시설 운영 = 총무 역량 + 복지 이해도의 교차점. 제 두 가지 경력이 만나는 포지션" |

## 에피소드 선택

1. **외주업체 8개사 통합 관리** — 식당/어린이집도 결국 외주 위탁 운영. 관리 프로세스 동일.
2. **ERP 도입 풀사이클** — 신사옥 기획 업무의 "새 시스템 구축" 역량 증명.
3. **예산 3.6억 집행·정산** — 비용 정산/예산 관리 직접 매칭.
