## #4 Positioning
- "총무"가 아니라 **"인프라 예산 관리 + ERP 구축"** 경험자로 프레이밍
- 3.6억 예산 직접 집행/통제 → 손익관리 역량
- ERP 회계계정 체계 설계 (여신/채무/이자) → 관리회계 시스템 이해
- 외주 8개사 계약/정산 총괄 → 전사 프로세스 관리
- 비용 30% 절감 + 재계약 500만원 절감 → 경영 효율화 마인드
- 소규모 조직에서 혼자 다 돌린 적응력 → 빗썸의 빠른 환경에 부합

## #5 Weak Points
- 경영기획 직접 경험 없음, 경력 3년 (5년 미달)
  → 방어: 현장에서 예산/정산/프로세스를 직접 돌린 실행력. ERP 도입 = 경영기획의 하위 레벨 경험
- 재무/회계 자격증 없음
  → 방어: ERP에서 여신 회계계정 직접 설계한 실무 경험으로 대체
- 정규직 공고 (계약직 전략과 불부합)
  → 수습 3개월 존재. 합격 시 판단.
