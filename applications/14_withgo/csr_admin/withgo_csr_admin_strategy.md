## #4 Positioning

**핵심 포지셔닝**
> "비영리 법인에서 3년간 예산 3.6억 집행·통제 + ERP 회계모듈 도입 PM + 사회복지사 1급 = 행정·회계 즉시 투입 가능"

**차별점**
- 비영리 현장 경험 + 학문적 배경(석사) 동시 보유
- ERP 도입 경험 → 디지털 행정 적응 능력 입증
- 사회복지사 1급 → 기관 미션·사업 이해도 높음 (단순 행정직과 차별)

**약점 방어 로직**
- CSR 직접 경험 부족 → "비영리 총무로서 외부 협력사·자원봉사 행정 지원 수행. CSR의 행정 지원 역할과 본질적으로 동일"
- 보조금 사업 경험 부족 → "예산 집행·증빙·정산 프로세스는 동일. 보조금 규정 숙지에 빠르게 적응 가능"
- 회계 직접 담당 아님 → "ERP 도입 PM으로 회계모듈 설계·검증 참여. 프로세스 이해도 실무자 수준"
