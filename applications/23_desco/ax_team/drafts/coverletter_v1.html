경영지원 · AI Transformation 지원
비효율을 발견하면 끝까지 결과물로 만드는 사람입니다. 3년간 경영지원 실무를 하면서 반복 업무와 수작업 의존의 문제를 직접 체감했고, 이를 데이터 구조화와 프로세스 재설계로 해결해왔습니다. AX TEAM의 "업무 프로세스 AI 자동화" 미션은 제가 개인 프로젝트로 이미 실천하고 있는 방향과 정확히 일치합니다. 광명 거주자로서 KTX광명역 앞 GIDC 출퇴근도 최적입니다.
01
엑셀 수작업 예산 관리 → ERP 디지털 전환
문제 200여 개 예산 항목을 엑셀로 수기 관리. 부서별 집행 현황 파악에 수일 소요. 오류 발생 시 추적 불가.
해결 커스텀 ERP(Tibero DB/Nexacro) 도입 프로젝트를 단독으로 수행. 요구사항 정의 → 계정과목 200개 표준화 → SQL 데이터 정합성 검수 → 기능 검수까지 풀사이클.
결과 전사 예산·회계 시스템 디지털화 완료. 3.6억 원 예산 집행률 88.5% 달성. 데이터 기반 의사결정 가능한 환경 구축.
JD 매칭: "경영분석 지원 및 자동화" · "데이터를 읽고 맥락을 파악해 논리적으로 정리"
02
반복 서류 작업 → Claude + Python 자동화
문제 퇴직 후 구직 과정에서 기업별 이력서·자소서를 반복 작성하는 비효율을 체감. 자사양식(.hwpx) 수기 입력, 이메일 개별 발송 등.
해결 Claude(AI 에이전트) + Python으로 자동화 시스템을 직접 설계·구축. hwpx 자사양식 자동 입력(XML 파싱), HTML 이력서 자동 생성, SMTP 이메일 자동 발송, 지원 현황 데이터 파이프라인 운영.
결과 20여 건의 지원을 체계적으로 관리. 개인 프로젝트 수준이지만, AI 에이전트를 활용한 업무 자동화의 설계→구축→운영 전 과정을 직접 경험.
JD 매칭: "AI 에이전트 기반 자동화 솔루션 직접 구축·운영" · "비효율을 발견하면 끝까지 결과물로 만들어내는 집요함"
03
외주 유지보수비 과다 → 1차 대응 내재화
문제 네트워크·PC 장애 발생 시 매번 외주업체 호출. 단순 트러블슈팅에도 출장비 발생.
해결 반복 장애 패턴을 분석하고 1차 대응 매뉴얼 정리. 직접 트러블슈팅 수행 체계 구축.
결과 연간 유지보수 비용 30% 절감. 재계약 협상으로 추가 연 500만 원 절감.
JD 매칭: "반복 흐름 발견, 자동화 솔루션 구축" · "자기 주도적 문제 해결력"
채용공고에 "코딩 경험이 아니어도 좋습니다. 엑셀 매크로로 반복 업무를 줄인 경험, 프로세스를 정리해서 팀 효율을 높인 경험"이라는 문구가 있었습니다. 그 문장이 제 경력을 정확히 설명하고 있다고 느꼈습니다. 데이터를 구조화하고, AI로 프로세스를 재설계하는 실무자로서 AX TEAM에 기여하겠습니다.
