## 1. 약점 방어 (Defense Logic)

### D1. 게임/IT업계 경험 없음
> **인정 → 전환 → 근거**
- "게임업계 경험은 없습니다. 다만 예산 편성·집행·통제·보고의 구조는 산업을 가리지 않습니다."
- "비영리법인은 정부 보조금 정산 기준이 엄격해서, 오히려 일반 기업보다 예산 통제가 까다로운 환경이었습니다."
- "ERP 도입 시 개발사(Tibero/Nexacro)와 직접 소통한 경험이 있어, IT 환경 적응은 빠를 것입니다."
- **킬러 한마디**: "게임 도메인은 모르지만, 예산 시스템은 압니다. 도메인은 3개월이면 배우지만, 예산 풀사이클 경험은 3개월에 쌓이지 않습니다."

### D2. 회계 자격증 없음
- "자격증 대신 18개월 ERP 도입 PM을 직접 수행하며, 계정과목 표준화부터 검수까지 실무로 익혔습니다."
- "경제학 복수전공으로 회계·재무 기초 이론은 갖추고 있습니다."

### D3. 경력 공백 (2026.02~현재, 약 1개월)
- "3년간 재직하며 쌓은 역량을 정리하고, 다음 커리어 방향을 확실히 잡기 위해 의도적으로 시간을 가졌습니다."
- "공백 기간에도 구직 활동과 역량 개발을 병행하고 있습니다."

### D4. 제주 이전 괜찮은가?
- "네, 제주 근무에 대한 의향은 확고합니다. 이미 이사 준비를 고려하고 있습니다."
- "1종 대형 면허 보유, 12년 무사고 운전자라 제주 내 이동도 문제없습니다."

### D5. 12개월 계약직인데 동기가 진짜인가?
- "짧은 기간이라 오히려 더 빠르게 기여해야 한다는 생각입니다."
- "자소서에 기술한 대로, 3개월 내 독립 업무 수행 → 프로세스 개선 1건 제안이 목표입니다."
- "계약직이라는 형태에 관계없이, 팀에 실질적인 기여를 남기고 싶습니다."

### D6. 왜 전 직장을 퇴사했나?
- "3년간 1인 체제로 인프라 예산·IT·총무를 담당하며 많은 것을 배웠지만, 예산 관리를 더 깊이 전문화하고 싶다는 방향이 명확해졌습니다."
- "특히 ERP 도입 프로젝트를 마무리하면서, 시스템 기반 관리회계 영역에서 역량을 심화하고 싶었습니다."

## 2. 필살기 (Attack Logic)

### A1. 에피소드 — ERP 도입 PM (STAR)
- **S**: 비영리법인, Excel 의존 예산관리. 부서마다 계정과목 기준 상이 → 결산 시 며칠 소요
- **T**: 커스텀 ERP 도입 프로젝트의 예산·회계 영역 실무 전담
- **A**:
  - 전 부서 인터뷰 → 업무 흐름 문서화
  - 50여 개 계정과목 표준화 + 기준표 작성 → 부서 교육
  - 현업 요구 → 기술 요구사항 변환 (개발자 경험 활용)
  - 전년도 데이터로 시뮬레이션 검수
- **R**: 월말 집계 "며칠 → 당일", 집행률 88.5% 투명 관리, 시스템 기반 실시간 집계 전환

### A2. 에피소드 — 외주업체 8개 계약 재협상
- **S**: 복합기·전자계약·본인확인 등 기존 조건 연장 분위기
- **T**: 데이터 기반 비용 절감 실현
- **A**: 사용량 데이터 분석 → 절감 시뮬레이션 → 내부 팀장 동의 → 업체 win-win 조건 제시
- **R**: 연 500만 원 절감 + 관계 유지 + 내재화로 유지보수 30% 절감

### A3. 차별점 — "Why Me?"
1. **풀사이클 경험자**: 편성→집행→통제→보고→ERP 설계까지 한 사람이 다 해봄
2. **SQL 직접 쿼리**: 개발자 없이 DB에서 경영진 요구 데이터 즉시 추출
3. **번역자 역할**: 현업 ↔ 개발자 사이 요구사항 번역 (IT 개발 경험 + 예산 실무)
4. **비영리법인 예산 = 엄격한 감사 기준**: 정부 보조금 정산 경험 → 투명성·정확성 체질화
