## #3 Go/No-Go

### ✅ **STRONG GO**

**Go 근거:**
1. **계약직 1년** → 유저 전략(계약만료→퇴직금+실업급여) 완벽 부합
2. **ERP 경험 직결** → HighQ ≈ SAP 아키텍처. "ERP 도입 풀사이클" 이력이 SAP 활용 업무에 강력한 셀링포인트
3. **데이터 취합/보고서** → 3.6억 예산 집행 + 정산 보고 = 매출 데이터 취합/보고서와 본질 동일
4. **부서 조율 역할** → 총무 3년 = 전사 허브 역할. "지사 요청 접수/조율"과 정확히 겹침
5. **자격요건 낮음** → 전문학사 이상. 석사 소지자로서 오버퀄
6. **글로벌 기업** → Canon Japan 한국법인. 체계적 근무환경 기대
7. **강남 본사** → 광명에서 1호선→신분당선 or 2호선 환승, 1시간 내외

**약점 (방어 필요):**
- 영업 직접 경험 없음 → "총무로서 전사 요청 조율 = 내부 영업지원과 본질 동일" 프레임
- 신입 채용인데 경력자 → "계약직이라 즉전력 환영" + "ERP 경험이 온보딩 단축"

## #4 Positioning

### 메인 컨셉: "ERP 실전 경험 보유 즉전력 영업지원"

### 3축 포지셔닝
1. **SAP ≈ HighQ 실전 경험**: ERP 도입 풀사이클(요구정의→설계→검수) 경험자. SAP 전환 시 러닝커브 최소화
2. **데이터 기반 보고서 역량**: 3.6억 예산 집행 데이터 관리 + Excel Pivot/VBA 활용한 정량 보고서 작성
3. **전사 허브 경험**: 총무 3년간 8개 외주업체 + 전부서 요청 접수·조율 → "지사 요청 접수/조율" 업무에 즉시 투입 가능

### 약점 방어
| 예상 질문 | 방어 로직 |
|----------|----------|
| "영업 경험이 없는데?" | 총무로서 전사 요청 접수·조율 = 내부 영업지원. 외주 8개사 계약·정산·견적 관리 경험 |
| "왜 캐논코리아?" | 복합기 렌탈 계약·정산 관리 경험으로 이미징/프린팅 업계 프로세스 이해 + 글로벌 기업의 체계적 업무환경 |
| "석사인데 왜 신입 계약직?" | 실무 역량으로 증명하겠다는 의지. ERP·데이터 경험을 영업지원에 접목하고 싶음 |

### 지원 동기 핵심 서사
> 총무로서 **복합기 렌탈 계약·재계약 협상·정산을 직접 관리**했던 경험 → 이미징/프린팅 업계의 영업-고객 간 프로세스를 **고객사 실무자 관점에서** 체득. 그 경험을 이번엔 **제공자 측에서** 활용하고 싶다. ERP 도입 경험을 SAP 기반 영업지원에 접목.
