PROJECT 01
사회공헌 기관 운영 — 시니어 교육·복지의 백본
2023.01 → 2026.02
KDB시니어브리지아카데미를 운영하는 (사)함께만드는세상에서, 시니어 대상 교육·복지 사업이 원활히 돌아가도록 인프라 운영·외부기관 협업·행정 전반을 3년간 담당했습니다. 봉사활동 현장 운영에 직접 필요한 — 협력기관 관리, 현장 물류, 문서·정산, 커뮤니케이션 — 모든 역량을 이 현장에서 체득했습니다.
협력기관 관리
외주업체 8개사(소방·승강기·전기·방역·복합기 등) 계약·관리·정산 총괄. 재계약 협상으로 연간 유지보수 비용 30% 절감.
전사 커뮤니케이션 허브
전부서 요청 접수·조율의 단일 창구. 내부 직원 + 외부 협력기관 + 파트너까지 다양한 이해관계자와 일상적으로 소통.
현장 운영 지원
시니어 교육 프로그램 행사장 셋업, 물품 이동, 장비 점검 등 현장 밀착 지원. 체력 활동과 사무 행정을 병행.
예산 집행·문서 관리
인프라 예산 약 3.6억 원 직접 집행(집행률 88.5%). 계약서·발주서·정산서 등 행정 문서 전담.
CHALLENGE
외주업체 분산 관리 → 일관된 품질 유지 어려움
전부서 문의 폭주 → 우선순위 없이 대응
현장·사무 동시 수행 → 체력·시간 관리 필요
RESULT
외주 단일 관리 체계 구축 → 비용 30% 절감
FAQ 매뉴얼화 → 반복 문의 즉시 처리
현장/사무 루틴 최적화 → 3년간 안정 운영
예산 집행률 88.5% · 외주 비용 30% 절감 · 프로세스 100% 문서화
PROJECT 02
ERP 도입 — 조직 운영 체계의 디지털 전환
2023.07 → 2024.12
엑셀 수기 관리 환경에서 커스텀 ERP(High-Q)를 요구사항 정의부터 검수까지 단독으로 도입한 프로젝트입니다. 200여 개 계정과목 표준화, 데이터 마이그레이션, 사용자 교육까지 — 새로운 제도·시스템을 체계적으로 정리·실행하는 역량을 이 과정에서 입증했습니다.
Phase 1 — 현황 분석 & 요구사항 정의
현업 부서 인터뷰 → 기존 프로세스 분석 → 기능 요구사항 문서화. 200개 계정과목 재분류·표준화.
Phase 2 — 설계 & 데이터 마이그레이션
예산/회계 프로세스 재설계. 기존 엑셀 데이터 정합성 검증 및 마이그레이션. SQL 직접 작성.
Phase 3 — 검수 & 안정화
기능 테스트 → 수정 요청 → 재검증 사이클 반복. 관리자 권한으로 시스템 운영 및 사용자 문의 대응.
PM 단독 수행 · 계정과목 200개 표준화 · 결산 리드타임 2일 단축

현장 기동력

1종 대형면허 + 승합차(스타렉스) 운전 경험 + 12년 무사고. 전국 봉사활동 현장까지 직접 이동하고, 물품 운반하고, 행사를 셋업하는 데 거부감이 없습니다.

📋

행정·운영 즉전력

문서 작성, 예산 집행, 외부 기관 협업, 행사 운영 — 빗썸나눔의 '활동 지원 담당자'에 필요한 실무를 3년간 매일 수행했습니다. 온보딩 없이 즉시 투입 가능합니다.

🎓

사회복지 전문성

사회복지사 1급 + 숭실대 사회적기업전공 석사. 비영리 조직의 특수성을 이해하고, 복지 현장의 언어로 소통할 수 있는 기반입니다.
