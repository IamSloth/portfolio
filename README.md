# Apply Trae Agent (Legacy / Paused)

> The DOM-based automation approach reached its limits with complex dynamic UIs (popups, iframes, obfuscated classes).
> **Plan for Resumption**: Migrate to a **Vision-based AI Agent** (using GPT-4V, Stagehand, or Midscene) to handle UI interactions more robustly like a human.

## Project Overview
An autonomous agent attempt to apply for jobs (specifically Smilegate CSR) by parsing resumes and interacting with the browser.

## Key Outcomes
- Successfully implemented **Human-in-the-loop** browser connection via CDP.
- Solved **Naver Login** automation using event dispatching.
- Automated email parsing for Hackathon topics.
- Documented technical bottlenecks in `PROJECT_RETROSPECTIVE.md`.
