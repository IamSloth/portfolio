02 — Project Deep-Dive

PROJECT 01
ERP 도입 실무 전담 — 회계 프로세스 100% 전산화
2023.07 → 2024.12 · 18개월
엑셀 기반 장부 관리 환경에서 커스텀 ERP(High-Q, Tibero/Nexacro 기반)를 요구사항 정의부터 검수까지 실무 전반을 전담한 프로젝트입니다. 30여 개 회계계정 체계를 직접 설계하고, 엑셀→DB 데이터 마이그레이션·사용자 교육까지 수행했습니다. 더존 iCube는 마이그레이션 과정에서 다뤄본 경험이 있으며, 아마란스ID 또한 동일 계열 ERP로서 빠른 적응이 가능합니다. 현재 더존 iCube 실무 교육을 병행하며 빠른 투입을 준비하고 있습니다.
Phase 1 — Requirements & Account Design
현업 부서 인터뷰 → 기존 프로세스 분석 → 기능 요구사항 문서화. 30여 개 회계계정과목 체계 정의 및 표준화.
Phase 2 — Data Migration & Validation
엑셀 → Tibero DB 마이그레이션. SQL 기반 데이터 정합성 검증. 계정과목별 잔액 대사.
Phase 3 — Testing & Stabilization
기능 테스트 → 수정 → 재검증 사이클 반복. 시스템 안정화 및 사용자 교육. 회계 프로세스 100% 전산화 완료.
BEFORE
엑셀 장부 → 데이터 정합성 미보장
계정과목 비표준 → 결산 지연
수동 데이터 추출 → 분석 병목
AFTER
DB 기반 관리회계 → 실시간 데이터 접근
회계계정 체계 구축 → 결산 2일 단축
SQL 직접 추출 → 분석 속도 향상
실무 전담 18개월 · 30개 계정과목 설계 · 회계 프로세스 100% 전산화
PROJECT 02
예산 관리 & 감사 대응 — 3.6억 편성·집행·결산
2023.01 → 2026.01 · 3년
연 3.6억 규모의 인프라 예산을 단독으로 편성·집행·결산하며, 8개 외주업체의 계약·정산·관리를 총괄했습니다. 계정과목별 지출결의·집행·증빙 관리 전 과정을 직접 수행하고, 외부 감사 대응까지 전 주기를 1인 전담한 경험입니다.
Budget Planning & Execution
연간 3.6억 인프라 예산 편성. 30여 계정과목 기준 월별 집행 현황 분석 및 보고. 집행률 88.5% 달성.
Vendor & Contract Management
소방·승강기·복합기·네트워크 등 8개사 계약·정산 총괄. 재계약 협상으로 연 500만 원 절감.
Audit Response — Full Chain
사업협약부터 통장내역까지 전체 증빙 체계를 ERP 기반으로 정비하여 외부 감사에 대응.

감사 대응 증빙 체인

사업협약→ 예산계획→ 예산편성→ 지출결의→ 회계전표→ 지급전표→ 통장내역
집행률 88.5% · 유지보수 비용 -30% · 연 500만 절감 · 감사 적격
